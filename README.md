## changelog

### V1.0
* 创建和完成基本的浏览器兼容性测试
* 支持IE >= 8浏览器，鼠标滚轮滚动切换日历
* 添加了丰富的日期格式选择、输入和输出

### V1.1
* 停止了对DateTimePicker()方式的调用方式，必须new DateTimePicker()
* 添加了丰富的函数批量，如获取当前日期、时间
* 整理了event抛出，不会再出现event响应混乱问题
* 整理了文档
* demo支持在线调试
